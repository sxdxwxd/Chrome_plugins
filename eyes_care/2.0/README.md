# 护眼模式切换器

一个浏览器插件，用于一键切换页面到黑暗或护眼模式，保护您的眼睛健康。

## 功能特性

- 三种模式切换：默认模式、暗黑模式、护眼模式
- 自定义颜色设置：可以根据个人喜好调整每种模式的颜色
- 持久化存储：设置会自动保存，下次打开浏览器时仍然有效
- 支持Edge和Chrome浏览器

## 安装方法

### Chrome浏览器
1. 打开Chrome浏览器，输入 `chrome://extensions/` 并回车
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择插件所在的文件夹（包含manifest.json的文件夹）
5. 插件安装完成，右上角会出现插件图标

### Edge浏览器
1. 打开Edge浏览器，输入 `edge://extensions/` 并回车
2. 开启左上角的"开发者模式"
3. 点击"加载解压缩的扩展"
4. 选择插件所在的文件夹（包含manifest.json的文件夹）
5. 插件安装完成，右上角会出现插件图标

## 使用说明

1. 点击浏览器右上角的插件图标，打开弹出窗口
2. 在弹出窗口中选择想要的模式：
   - **默认模式**：不应用任何主题，保持页面原有样式
   - **暗黑模式**：将页面切换为深色主题，适合夜间使用
   - **护眼模式**：将页面切换为柔和的护眼主题，减少眼睛疲劳
3. 选择模式后，页面会立即应用相应的主题
4. 可以点击"保存设置"按钮保存当前的设置
5. 如果不满意当前设置，可以点击"恢复默认"按钮恢复到初始设置

## 自定义设置

1. 选择一种模式（暗黑模式或护眼模式）
2. 颜色设置面板会自动显示
3. 点击颜色选择器可以调整以下颜色：
   - 背景色：页面的背景颜色
   - 文字色：页面文字的颜色
   - 链接色：页面链接的颜色
   - 强调色：页面强调元素（如按钮、边框等）的颜色
4. 调整完成后，点击"保存设置"按钮应用新的颜色设置

## 技术说明

- 插件使用Chrome扩展API开发，兼容Edge浏览器
- 使用`chrome.storage.sync`存储用户设置，实现跨设备同步
- 使用内容脚本修改页面CSS样式，实现主题切换
- 支持动态调整主题颜色，实时预览效果

## 注意事项

1. 某些网站可能有复杂的样式，插件可能无法完全覆盖所有元素
2. 如果发现页面样式异常，可以切换回默认模式
3. 建议根据个人使用习惯和环境调整颜色设置
4. 定期休息，合理使用护眼模式，保护眼睛健康
